# MoguSync

MoguSync is a mobile app and Discord bot system for controlling shared music playback inside Discord voice channels.

The app works as a remote controller. Users log in with Discord, select a server and voice channel, then command the backend to connect the bot and manage the music queue.

## Main App Navigation

```text
Queue | Search | Servers | Library | Settings
```

### Queue

Displays the active session, current track, playback controls, vote skip, and upcoming queue.

### Search

Searches external music sources or accepts pasted links, then adds tracks to the queue.

### Servers

Handles Discord server selection, bot install state, voice channel selection, and active session discovery.

### Library

Displays local/server-hosted tracks from the MoguSync backend storage.

### Settings

Handles account settings, Discord integration, DJ role config, voting config, and queue limits.
